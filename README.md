# DAIICT-Hackathon

## Run Locally

Clone the project

```bash
  git clone https://github.com/kevalpanwala27/DAIICT-Hackathon
```

Go to the project directory

```bash
  cd DAIICT-Hackathon
```

Install dependencies in server

```bash
  npm install
```

Go to the client folder

```bash
  cd client
```

Install dependencies in client

```bash
  npm install
```

Start the client react project

```bash
  npm run start
  ```

Come back to server

```bash
  cd ..
```

Start Server using node or nodemon

```bash
  node app.js 
  or nodemon app.js
```
